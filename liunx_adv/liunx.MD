1.fopen打开流

FILE *fopen(corist char *pathname, const char *mode);


函数的功能
。打开一个文件，并返回一个流(文件指针)
函数的参数

pathname:要打开的文件名，例如"/home/linux/1.txt”
mode:打开文件的方式

b  binary的缩写，表示以二进制的形式打开文件，操作的是二进制流
r或rb 打开只读文件，该文件必须存在，文件不存在报错。
r+或r+b 打开可读写的文件，该文件必须存在。
wwb 打开只写文件，若文件存在则文件长度清为0，即会擦些文件以前内容。若文件不存在则建立该文件。
w+或w+b或wb+ 打开可读写文件，若文件存在则文件长度清为零，即会擦些文件以前内容。若文件不存在则建立该文件。
aab  以附加的方式打开只写文件。若文件不存在，则会建立该文件，如果文件存在，写入的数据会被加到文件尾，即文件原先的内容会被保留。
a+或a+b或ab+ 以附加方式打开可读写的文件。若文件不存在，则会建立该文件，如果文件存在，写入的数据会被加到文件尾后，即文件原先的内容会被保留。
