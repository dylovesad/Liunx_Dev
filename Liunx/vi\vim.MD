vi/vim 文本地址 //进入模式(list)

可以使用(nano)

/*** 快捷键 ***/ 
h   左移
j   下移
k   上移
l   右移

w   跳到下一个单词开头
b   跳到上一个单词开头
0   跳到行首
$   跳到行尾
gg  跳到文件第一行
G   跳到文件最后一行

i   在光标前插入
I   跳到行首插入
a   在光标后插入
A   跳到行尾插入
o   在当前行下方新建一行
O   在当前行上方新建一行

x   删除光标所在字符
dd  删除整行
dw  删除一个单词
D   从光标删到行尾
u   撤销（undo）
Ctrl + r  恢复撤销（redo）

yy  复制当前行
yw  复制一个单词
p   粘贴到光标后
P   粘贴到光标前

/关键词   向下搜索
n        查找下一个
N        查找上一个
:%s/旧词/新词/g   全文替换

:w     保存
:q     退出
:wq    保存并退出
:q!    强制退出（不保存）
ZZ     快速保存并退出（不用输冒号）

Ctrl + f  向下翻一页
Ctrl + b  向上翻一页
.        重复上一次操作
>>       向右缩进
<<       向左缩进
